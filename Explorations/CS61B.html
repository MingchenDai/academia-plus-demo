Lecture 1

Introduction to Java

In Java:

  • All code must be part of a class.
  • Classes are defined with public class Classname
  • The code we want to run must be inside public static void main(String[] args).

Java is a language caring much about object-oriented programming with strict requirements:

  • Every Java file must contain a class declaration.
  • All code lives inside a class even helper functions and golbal constants.
  • To run a Java program,you should typically define a main method using public static void main(String[] args).

In Java, to define a function we use public static.

In most conditions this keyword is optional.

Defining and Using Classes

  • An object is an instance of any class.
  • Each class has its own variables known as instance variables or non-static variables which mush be declared inside the class.
  • Those methods with out static keyword are called instance methods or non-static methods.

Classes can be instantiated and instances can hold data. In such classes if we want to call a certain method, we should firstly instantiate an object.

Variables and methods of a class are also called menbers of a class which can be accessed using dot notation.
